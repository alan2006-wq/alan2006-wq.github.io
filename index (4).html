Project Overview

This tool helps you analyze and design trusses for the EK301 project. The goal is to design a truss capable of supporting a given load while minimizing cost.

Key Assumptions

  • Structure is modeled as a pin-jointed 2D truss
  • Tensile strength of members is practically infinite
  • Joint strength is practically infinite
  • Dominant failure mechanism is buckling of compression members

Design Specifications

Joint-to-joint span: 6–14 inches
Truss span: 26–30 inches
Load to pin (horizontal): 9–11 inches
Vertical offset (Δy): 0–2 inches
Minimum load: 32 oz (2 lb)

Analysis Method

The truss analysis uses equilibrium equations at each joint to solve for unknown member forces and support reactions.

Matrix Approach

The system can be expressed as:

$$\mathbf{A} \cdot \mathbf{T} + \mathbf{L} = \mathbf{0}$$

Where:

  • A is the equilibrium matrix (2J × M+3)
  • T is the unknown force vector (member tensions and reactions)
  • L is the known load vector

Solving for unknowns:

$$\mathbf{T} = -\mathbf{A}^{-1} \mathbf{L}$$

Linearity Principle

Member forces scale linearly with applied load:

$$T_m = R_m \times W_\ell$$

Where \(R_m\) is independent of load magnitude.

Critical Member Identification

The critical member is the compression member that fails at the lowest applied load.

For each compression member:

$$W_{failure,m} = \frac{-P_{crit,m}}{R_m}$$

The minimum value identifies the critical member:

$$W_{failure} = \min_{\{m \mid R_m < 0\}} \left( \frac{-P_{crit,m}}{R_m} \right)$$

Key Formulas

Simple Truss Relation

$$M = 2J - 3$$

Where M is the number of members and J is the number of joints.

Cost Function

$$C = C_L L_{total} + C_J J$$

Where:

  • \(C_L = 1\) unit/inch
  • \(C_J = 10\) units/joint
  • \(L_{total}\) = total member length (inches)
  • \(J\) = number of joints

Buckling Strength

$$F = C \left(\frac{L_0}{L}\right)^\alpha \pm U_{fit}$$

Spring 2026 parameters:

  • \(C = 37.5\) oz (fit coefficient)
  • \(L_0 = 10\) in (reference length)
  • \(\alpha = 2\) (power law exponent)
  • \(U_{fit} = 10\) oz (total uncertainty)

Load-to-Cost Ratio

$$\text{Efficiency} = \frac{W_{failure}}{C}$$

Units: oz/$ (higher is better)
